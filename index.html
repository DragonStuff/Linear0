Linear0

Linear0 is a program made by volunteers that assists with bypassing the extremely restrictive firewall employed by the DET/DEC. The current version is: 0.2.1

How can I use this?

Download Linear0 using the link above. Simply open Linear0.exe and type in your DEC credentials. The application will launch and automatically configure Internet Explorer, Google Chrome and Windows Store Applications.

You may need to configure Mozilla Firefox (or other applications) in the following way:

Options -> Advanced -> Network -> (Connection) Settings

  • Host: 127.0.0.1
  • Port: 8118
